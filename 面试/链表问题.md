# 1、在一次遍历中，怎样发现单个链表的中间元素?  

我们可以设置两个指针变量，node *p ,node *q，p每次移动两个位置，q每次移动一个位置，当p.next==null时候，这时候q指针就到了链表的中间位置。     
```C
void search(node *head,node *mid)
{
    node *temp=head;
    while(head->next->next!=NULL)
    {
        head=head->next->next;
        temp==temp->next;
        mid=temp;

    }
}

```
