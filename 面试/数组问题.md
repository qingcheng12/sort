# 7、如何从一个数组中删除重复元素？

只去看它后面的元素是否有和它相同，若有相同则把该元素移动到当前位置的下一个位置即可。然后继续向后找看是否有相同的元素。  

```C
int removeDuplicates(int* nums, int numsSize) {
    int i = 1;
    if(0 == numsSize)
    {
        i = 0;
    }
    else
    {
        for(int j = 1; j < numsSize; j++)
        {
            if(nums[j] != nums[i])
            {
                i++;
                nums[i] = nums[j];
            }
        }
    }
    return i;
}
```
